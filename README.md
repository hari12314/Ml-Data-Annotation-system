# ML Data Annotation & Quality Evaluation System

## Overview
This project is a Streamlit-based application designed to simulate real-world machine learning data annotation workflows. It supports annotation for text, image, and audio data, along with quality evaluation and productivity tracking.

## Features
- Text annotation (Sentiment / Intent classification)
- Image annotation (Object labeling)
- Audio annotation (Emotion / Language tagging)
- Quality validation and error tracking
- Dashboard for monitoring performance and productivity

## Tech Stack
- Python
- Streamlit
- Pandas
- SQLite
- Plotly

## Use Case
Simulates ML data pipelines used in AI systems for:
- Data labeling
- Quality assurance
- Model preparation
